# MyWardrobe

Το MyWardrobe είναι μια εφαρμογή μόδας που βοηθά στην οργάνωση της ντουλάπας σας, τη δημιουργία συνδυασμών ρούχων (outfits) και την παρακολούθηση φίλων για στυλιστική έμπνευση.

## Οδηγίες Εγκατάστασης και Χρήσης

**Σημείωση:**
Η παρούσα εφαρμογή απαιτεί τυπική παραμετροποίηση Android. Εάν διαθέτετε το αρχείο εγκατάστασης (APK), απλώς εγκαταστήστε το στη συσκευή σας. Εάν επιθυμείτε τη μεταγλώττιση (build) από τον πηγαίο κώδικα, ακολουθήστε τις παρακάτω οδηγίες.

### 1. Προαπαιτούμενα
Για την εκτέλεση της εφαρμογής από τον πηγαίο κώδικα, απαιτούνται τα εξής:
*   Εγκατάσταση του **Flutter SDK** στον υπολογιστή σας: [Οδηγίες Εγκατάστασης](https://docs.flutter.dev/get-started/install).
*   Συσκευή Android ή Εξομοιωτής (Emulator). Η συσκευή πρέπει να διαθέτει **Google Play Services** για τη λειτουργία των υπηρεσιών Firebase.
*   **Σημαντικό**: Απαιτείται η ύπαρξη του αρχείου `google-services.json` στον φάκελο `android/app/`. Το αρχείο αυτό περιέχει τα κλειδιά παραμετροποίησης του Firebase.

### 2. Βήματα Εγκατάστασης
1.  Αποσυμπιέστε ή κλωνοποιήστε (clone) τον φάκελο του έργου.
2.  Ανοίξτε τερματικό (terminal) εντός του φακέλου του έργου (`MyWardrobe`).
3.  Εκτελέστε την εντολή για τη λήψη των βιβλιοθηκών:
    ```bash
    flutter pub get
    ```
4.  Συνδέστε τη συσκευή Android ή εκκινήστε τον εξομοιωτή.
5.  Εκτελέστε την εφαρμογή:
    ```bash
    flutter run
    ```

### 3. Χρήση της Εφαρμογής

#### Διαχείριση Λογαριασμού
*   **Είσοδος (Sign In)**: Για να αποκτήσετε πρόσβαση στην εφαρμογή, δημιουργήστε έναν νέο λογαριασμό ή συνδεθείτε.
    *   *Demo Λογαριασμός*: Μπορείτε να χρησιμοποιήσετε τα εξής στοιχεία για άμεση δοκιμή:
        *   **Email**: `babisstokas@gmail.com`
        *   **Password**: `pyxlaxforever`

#### Ψηφιακή Ντουλάπα (Wardrobe)
*   **Προσθήκη Ρούχων**: Πατώντας το εικονίδιο της κάμερας, μπορείτε να φωτογραφίσετε τα ρούχα σας. Η εφαρμογή προσφέρει:
    *   **Αυτόματη Αφαίρεση Φόντου**: Απομονώνει το ρούχο από το περιβάλλον για καθαρή εικόνα.
    *   **Αυτόματη Κατηγοριοποίηση**: Αναγνωρίζει αν είναι μπλούζα, παντελόνι κλπ.
*   **Επεξεργασία Αντικειμένων**: Μπορείτε να προσθέσετε λεπτομέρειες όπως Μάρκα (Brand), Τιμή, Μέγεθος και σημειώσεις για κάθε ρούχο.

#### Δημιουργία Συνόλων (Outfits)
*   **Mix & Match**: Στην καρτέλα "Outfits", μπορείτε να επιλέξετε ρούχα από την ντουλάπα σας και να τα συνδυάσετε σε έναν καμβά.
*   **Αποθήκευση**: Αποθηκεύστε τους αγαπημένους σας συνδυασμούς για μελλοντική αναφορά.

#### Κοινωνική Δικτύωση (Social)
*   **Αναζήτηση Χρηστών**: Στην αρχική οθόνη, χρησιμοποιήστε την μπάρα αναζήτησης για να βρείτε φίλους (π.χ. αναζητήστε "test" ή "nikiforos").
*   **Follow/Unfollow**: Ακολουθήστε χρήστες για να βλέπετε τα σύνολα που δημιουργούν.
*   **Προφίλ Φίλων**: Επισκεφθείτε το προφίλ άλλων χρηστών για να δείτε τη συλλογή των outfits τους και να εμπνευστείτε.

## Τεχνικές Απαιτήσεις

*   **Έκδοση Android SDK**:
    *   **Minimum SDK**: 21 (Android 5.0 Lollipop)
    *   **Target SDK**: Τελευταία σταθερή έκδοση (Android 14 / API 34)
    *   **Απαιτούμενο Image**: Android Emulator με **Google APIs** (απαραίτητο για χάρτες, πιστοποίηση χρήστη και ειδοποιήσεις).

*   **Εξαρτήσεις (Dependencies)**:
    *   Firebase (Authentication, Firestore, Storage)
    *   Camera & ML Kit (για λειτουργίες όρασης υπολογιστή)
