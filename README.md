# MyWardrobe

Το MyWardrobe είναι μια εφαρμογή μόδας που βοηθά στην οργάνωση της ντουλάπας σας, τη δημιουργία συνδυασμών ρούχων (outfits) και την παρακολούθηση φίλων για στυλιστική έμπνευση.

## Εγκατάσταση

1. Εγκαταστήστε το Flutter SDK
2. Αντιγράψτε το `google-services.json` στο `android/app/`
3. Τρέξτε:
   ```bash
   flutter pub get
   flutter run
   ```

### Χρήση της Εφαρμογής

#### Διαχείριση Λογαριασμού
*   **Είσοδος (Sign In)**: Για να αποκτήσετε πρόσβαση στην εφαρμογή, δημιουργήστε έναν νέο λογαριασμό ή συνδεθείτε.
    *   *Demo Λογαριασμός*: Μπορείτε να χρησιμοποιήσετε τα εξής στοιχεία για άμεση δοκιμή:
        *   **Email**: `babisstokas@gmail.com`
        *   **Password**: `pyxlaxforever`

#### Ψηφιακή Ντουλάπα (Wardrobe)
*   **Προσθήκη Ρούχων**: Πατώντας το εικονίδιο της κάμερας, μπορείτε να φωτογραφίσετε τα ρούχα σας. Η εφαρμογή προσφέρει:
    *   **Αυτόματη Αφαίρεση Φόντου**: Απομονώνει το ρούχο από το περιβάλλον για καθαρή εικόνα.
    *   **Αυτόματη Κατηγοριοποίηση**: Αναγνωρίζει αν είναι μπλούζα, παντελόνι κλπ. και τι χρώμα
είναι.
*   **Επεξεργασία Αντικειμένων**: Μπορείτε να προσθέσετε λεπτομέρειες όπως Μάρκα (Brand), Τιμή, Μέγεθος, Μήνας αγοράς, Συχνότητα Χρήσης και σημειώσεις για κάθε ρούχο.

#### Δημιουργία Συνόλων (Outfits)
*   **Mix & Match**: Στην καρτέλα "Outfits", μπορείτε να επιλέξετε ρούχα από την ντουλάπα σας και να τα συνδυάσετε σε έναν καμβά.
*   **Αποθήκευση**: Αποθηκεύστε τους αγαπημένους σας συνδυασμούς για μελλοντική αναφορά.

#### Κοινωνική Δικτύωση (Social)
*   **Αναζήτηση Χρηστών**: Στην αρχική οθόνη, χρησιμοποιήστε την μπάρα αναζήτησης για να βρείτε φίλους (π.χ. αναζητήστε "test" ή "nikiforos").
*   **Follow/Unfollow**: Ακολουθήστε χρήστες για να βλέπετε τα σύνολα που δημιουργούν.
*   **Προφίλ Φίλων**: Επισκεφθείτε το προφίλ άλλων χρηστών για να δείτε τη συλλογή των outfits τους και να εμπνευστείτε. (διαθέσιμα outfits στο testuser), δυνατότητα για like σε outfits φίλων και αποστολή notification από τους φίλους στον demo λογαριασμό.
